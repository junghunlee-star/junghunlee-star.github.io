# GitHub Pages Viewer

Upload this folder as-is to GitHub Pages.

- Entry page: `index.html`
- LCC data: `assets/260408_voidbox.lcc`
- Controls: `WASD` move, `R/F` up-down, drag to look

If your GitHub Pages URL is `https://<user>.github.io/<repo>/`, this viewer opens at:

`https://<user>.github.io/<repo>/`
